## 🌟 What is Spyrochat?

Spyrochat is a social messaging platform where you connect, share, and get rewarded for your engagement. Enjoy real-time chat, leaderboards, wallet features, status sharing, and a trusted community built around privacy and fairness.

---

## ✨ Features

**💬 Chat & Connect** — Message friends, build communities, and stay close in real time.

**💰 Wallet & Rewards** — Earn from your engagement from day one. No follower thresholds, no waiting periods — just complete basic verification and you're in.

**🔒 Privacy First** — Your private messages are never scanned or monitored. End-to-end encryption is available, and what you say in private stays private.

**📖 Personal Journal** — Set any post to Private and use it as your personal diary. Only you can see it, forever.

**🛡️ Verified Trust** — A three-tier verification system keeps the platform safe and authentic, with increasing perks at each tier.

**⏳ Content That Lasts** — Unlike platforms that delete stories after 24 hours, you control how long your content stays — up to forever on Tier 3.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | GoLang, PHP |
| Database | Laravel (Eloquent ORM) |
| Templating | Symfony (Twig) |
| HTTP / SPA | PhpSPA |
| Frontend | JavaScript, jQuery, CSS, Tailwind |
| Real-time | Go (WebSockets, goroutines) |
| Large Uploads | Go (streaming, compression) |

Go handles the heavy lifting — WebSockets, large file uploads, video processing, and complex concurrent operations. PHP handles the web layer, with Laravel for robust database management and Symfony/Twig for server-side rendering.
